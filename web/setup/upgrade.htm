Upgrading OpenEats

With OpenEats 1.1 a new upgrade script was introduced to help automate some of the task with OpenEats. Follow the following directions to upgrade OpenEats using the new upgrade script.
  1. Download the latest release from Soureforge
  2. Unzip/untar the downloaded file
  3. Create a backup of your current OpenEats installation by either copying your current OpenEats directory to another location or zipping up the current OpenEats directory
  4. Backup your database for OpenEats
  5. Copy all the contents of the new version of OpenEats that you just unzipped/untar to where you have OpenEats installed. If you are using a Mac, you will want to use the ditto command to do this because copy/paste a folder in Macs does not work the same as it does on Windows/Linux. To do this launch the Terminal.app found under the Utilites folder under Applications. Type the command ditto new_OpenEats_directory/ current_OpenEats_directory/
  6. Launch your browser and point it to your OpenEats URL with /web/setup/upgrade.php. For example if the URL you normaly access OpenEats at is http://localhost/oe You would use http://localhost/oe/web/setup/upgrade.php If your current URL to access OpenEats is http://oe.localhost You would use http://oe.localhost/oe/setup/upgrade.php
  7. Remove the cahce directory under the main OpenEats directory prior to trying to access your site.

Manual Upgrade

If for some reason the automated way does not work perform the following steps to upgrade OpenEats.
  1. Perform steps 1-5 from above
  2. Update your database. Run the appropriate sql file under the data/sql/upgrade folder under where you have installed OpenEats, using phpMyAdmin or any other SQL client. For example if you are upgrading from version 1.0 to version 1.1 use the upgrade_1.0_to_1.1.sql file, If there is not an sql file for your version then there was not any database changes and this step is not needed If you are upgrading from 1.o to 1.3 you may need to run three seperate SQL files.
  3. Edit the file apps/OpenEats/config/app.yml found under where you installed OpenEats. Change the version: 1.x to the version you just upgraded to. For example if the file said version: 1.0 and you upgraed to 1.1 change the 1.0 to 1.1
