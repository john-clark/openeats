Markdown Syntax


Markdown is a simple easy to use formatting syntax that allows you to produce fancy HTML elements with out having to learn HTML. Some of the basic syntax you can use is;

Phase Emphasis

To make words italic put one * in front of the word and one at the end for example *italic* would make the word italic appear italic. To make words bold put two * in front of the word and at the end for example **bold** would make the word bold appear bold

List

To create numbered list just enter

1. Item One

2. Item Two

To create unordered list just enter

* Item One

* Item Two


Headers


To create a headers add a "#" symbol in front of the header you want to create. For example to create a H1 header you add one # in front of a line, for a H2 header add two #'s.


Links


To add links to other pages or other recipes found on Openeats use this syntax [word you want linked] (link) This is [an example](http://example.com/ "Title") inline link. [This link](http://example.net/) has no title attribute.
